## How to - manage files and directories

```shell
pwd					                # display path of current location
ls					                # list contents of current folder
ls -a				                # include hidden folders
cd					                # change directory
clear				                # clear console
mkdir				                # Create a directory
touch				                # Create a file
rm					                # Remove file
mv					                # Move file: mv <file> <loc>
rmdir				                # Remove EMTPTY directories
rm -R				                # Remove all files and directory
open .				                # open current directory
```
