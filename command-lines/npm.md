## How to - use npm

```shell
npm init                        # create package.json
npm i -D <name>                 # install dev dependancy
npm i <name>                    # install dependancy
npm remove <name>               # uninstall package
npm uninstall <name>            # uninstall package
npm i -O                        # install optional dependancy
npm i --no-save                 # no dependancy
```
